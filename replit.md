# Community Wash for Life

## Overview
A static HTML/CSS website for Community Wash for Life, a charity organization focused on providing clean water, sanitation, and hygiene (WASH) solutions to underserved communities.

## Project Structure
- `index.html` - Main homepage
- `about.html` - About page
- `donate.html` - Donation page
- `gallery.html` - Photo gallery
- `programes.html` - Programs page
- `styles.css` - Main stylesheet
- `images/` - Image assets
- `server.py` - Simple Python HTTP server for development

## Technology Stack
- HTML5
- CSS3
- Bootstrap 5.3.2
- Font Awesome 6.4.0
- AOS (Animate On Scroll) library
- Python (for development server)

## Running the Project
The project uses a simple Python HTTP server on port 5000:
```
python server.py
```

## Deployment
This is a static site that can be deployed using Replit's static deployment option with the root directory as the public directory.
